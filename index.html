About

What is Bread?

Bread is a text editor currently under HEAVY development being written in Elixir. It aims to be open, fun, and flexible, allowing for high levels of customization by the users using the Elixir language itself.

What is the technology behind Bread?

Bread leverages any web-browser for it's UI, with plans on creating a standalone binary that will essentially just load a Webkit instance and display the Bread page. To do this, we have created a library, Butter. To find out more about how Butter works, please visit that page.
